## Requirements

- **Go 1.25+** (required to install coverctl; also needed for Go project coverage)
- **Git** (for diff-based coverage features)

:::tip[Multi-Language Support]
coverctl supports 15 languages: Go, Python, JavaScript/TypeScript, Java, Rust, C#, C/C++, PHP, Ruby, Swift, Dart, Scala, Elixir, and Shell. It auto-detects your project type during initialization.
:::

## Install from Source

The recommended way to install coverctl is using `go install`:

```bash
go install github.com/felixgeelhaar/coverctl@latest
```

This installs the latest release to your `$GOPATH/bin` directory.

## Install a Specific Version

```bash
go install github.com/felixgeelhaar/coverctl@v1.13.0
```

## Verify Installation

```bash
coverctl version
```

You should see output like:

```
coverctl v1.13.0 (40be00f) built 2026-03-01
```

## Shell Completions

coverctl supports shell completions for bash, zsh, and fish.

<Tabs>
  <TabItem label="Bash">
    ```bash
    # Add to ~/.bashrc
    eval "$(coverctl completion bash)"
    ```
  </TabItem>
  <TabItem label="Zsh">
    ```bash
    # Add to ~/.zshrc
    eval "$(coverctl completion zsh)"
    ```
  </TabItem>
  <TabItem label="Fish">
    ```bash
    # Add to ~/.config/fish/config.fish
    coverctl completion fish | source
    ```
  </TabItem>
</Tabs>
